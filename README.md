🧠 DSA Visualiser

An interactive platform to read, understand, and visualize Data Structures & Algorithms — built for students who learn best by seeing concepts in action.
📌 About the Project
DSA Visualiser is an educational web application that lists a wide range of Data Structures and Algorithms in one place. Each algorithm comes with:

📖 A clear explanation of how it works
💡 Step-by-step logic breakdown
📊 Visual representation of the process
⏱️ Time & Space Complexity analysis

Whether you're preparing for placements, revising for exams, or just learning DSA — this tool makes concepts easier to grasp.

🚀 Getting Started
Prerequisites
Just a browser — no installation needed for the live site.
To run locally:
bash# Clone the repository
git clone https://github.com/abhinandansharma11/DSA-VISUALISER.git

# Navigate to the project directory
cd DSA-VISUALISER

# Open index.html in your browser
open index.html
